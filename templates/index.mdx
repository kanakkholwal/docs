This site is a centralized place for all reusable templates and components. Each section is designed to help you quickly understand, use, and extend the templates.

## What You’ll Find

- ✅ Simple setup guides  
- ✅ Clear usage examples  
- ✅ API and config references  
- ✅ Consistent structure across all docs  

## How to Use

1. Browse the sidebar to explore templates  
2. Follow the instructions on each page  
3. Copy, modify, and integrate as needed  

## Contributing

To add or update a template:

1. Create a new `.mdx` file under `templates/` in the [github.com/kanakkholwal/docs](https://github.com/kanakkholwal/docs).  
2. Follow the existing format  
3. Open a pull request with your changes.
4. Ensure your changes are well-documented and tested.

That’s it — quick, clean, and useful. If you have any questions, feel free to reach out to the maintainers.
